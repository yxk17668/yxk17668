Git --changes not staged for commit

еҮәзҺ°зҡ„ж“ҚдҪң

жӯЈеёё git add . пјҢз„¶еҗҺgit commit

еҗҺд»Қз„¶жҠҘ changes not staged for commit й”ҷиҜҜпјҢе№¶дё”дёҠдј еҲ°д»“еә“зҡ„ж–Үд»¶еӨ№жҳҜз©әзҡ„

и§ЈеҶіж–№жЎҲ

  1. е…ҲејәиЎҢеҲ йҷӨcloneжқҘзҡ„зӣ®еҪ•дёӢзҡ„ .git ж–Үд»¶еӨ№

еҲ йҷӨж–№ејҸпјҡеңЁиҜҘзӣ®еҪ•дёӢжү“ејҖе‘Ҫд»ӨиЎҢе·Ҙе…·пјҢжү§иЎҢ rd/s/q .gitе‘Ҫд»Ө

  1. еӣһеҲ°д»“еә“ж №зӣ®еҪ•еҲ йҷӨд»“еә“дёӯзҡ„з©әж–Үд»¶еӨ№

вҖӢ 1пјүgit rm -r вҖ“cached вҖңthemes/materailвҖқ

вҖӢ 2пјүgit commit -m вҖңremove empty folderвҖқ

  1. еңЁд»“еә“ж №зӣ®еҪ•йҮҚж–°жҸҗдәӨд»Јз Ғ

вҖӢ 1пјүgit add .

вҖӢ 2пјүgit commit -m вҖңrepushвҖқ

вҖӢ 3пјүgit push origin master
